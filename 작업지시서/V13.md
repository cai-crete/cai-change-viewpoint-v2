# 작업지시서 V13

**작성일:** 2026-04-02
**목적:** 등각투영(정면/측면) vs 사선(코너) 뷰 생성 시점의 [PHASE 2] / [PHASE 3] 치중도 로직 차이 분석

---

## 1. 프롬프트 로직 및 의존도 차이 분석

현재 `App.tsx`의 렌더링 로직(`handleGenerate`)을 분석한 결과, **사용자가 선택한 시점(각도)에 따라 PHASE 2(시각 도면)와 PHASE 3(텍스트 규칙)의 치중도가 근본적으로 달라집니다.**

### A. 등각투영 뷰 (예: 09:00 Left, 03:00 Right, 12:00 Rear)
- **제공되는 이미지:** 원본 사진(IMAGE 1) + PHASE 2 도면 **1장** (IMAGE 2: 해당 면의 정사영 도면)
- **치중도 현황:** **[균형 ~ PHASE 3 약간 우세]**
  - AI는 1장의 도면만 참고하면 되므로, PHASE 3의 텍스트 규칙(서비스 도어 추가 등)을 해석하고 반영할 "자유도(인지적 여유)"가 있습니다.
  - 도면(형상)과 텍스트(설명)가 충돌하더라도 1D 평면 생성이므로 텍스트 논리가 개입될 여지가 큽니다.

### B. 사선 / 코너 뷰 (예: 04:30 Front-Right, 07:30 Front-Left)
- **제공되는 이미지:** 원본 사진(IMAGE 1) + PHASE 2 도면 **2장** (IMAGE 2: 정면 뷰, IMAGE 3: 측면 뷰)
- **프롬프트 텍스트:** *"두 도면 모두 PHASE 2에서 계산된 엄격한 기하학적 청사진이다. 2소실점 투시도로 두 면을 모두 정확히 통합(integrate)해야 한다."*
- **치중도 현황:** **[PHASE 2 압도적 편중]**
  - AI는 전혀 다른 시점의 두 정사영 도면을 3D 코너 뷰(Perspective)로 매칭하여 합성해야 하는 **엄청난 기하학적 제약**을 받습니다.
  - 이 과정에서 AI의 연산력은 **"두 이미지를 시각적으로 안 깨지게 붙이는 것(PHASE 2 준수)"**에 올인됩니다.
  - 결과적으로 PHASE 3의 섬세한 텍스트 논리("측면 벽에는 창문을 없애라")는 이미지 병합 제약에 밀려 완전히 **무시되거나(Bias) 원본을 그대로 뱉어내는 오류**를 범하게 됩니다.

## 2. 결론 및 원인 진단

**결론: 사선의 코너 뷰를 생성할 때, 이미지 퍼즐(PHASE 2 도면 2장)을 맞추는 시각적 구속력이 너무 강해져서 텍스트 로직(PHASE 3)이 작동을 멈춥니다.**

06:00/03:00 뷰는 도면 1장만 따르므로 텍스트 반영이 가능하지만, 04:30/07:30은 도면 2장을 강제로 결합하려다 보니 **PHASE 2의 시각 정보에 완전히 록인(Lock-in)**되어 발생한 현상입니다.

---

## 3. 해결 방안 (3가지 제안)

### 제안 1. 코너 뷰의 기하학 압박 완화 (보조면 시각 제공 중단)
- 사선 뷰(04:30) 생성 시, "정면 도면(Front)" 1장만 IMAGE 2로 제공하고, "측면 도면(Right)"은 시각 데이터에서 제외합니다. 측면의 형태는 오직 PHASE 3의 텍스트(Layer C) 논리로만 그리게 합니다.
- **효과:** 코너 결합의 강박이 풀려 텍스트(PHASE 3) 지시가 살아남.

### 제안 2. 코너 보조면 프롬프트 가중치 하향 지시
- 코너 뷰 생성 시 IMAGE 2와 IMAGE 3을 모두 주되, 명령어를 바꿉니다.
  *(예: "IMAGE 2(정면)는 완벽히 준수하되, IMAGE 3(측면)은 대략적인 참고만 하고 상세 내용은 Step 3의 텍스트 규칙을 최우선으로 그려라.")*
- **효과:** 구조 변경 없이 프롬프트 우선순위만 조정하여 밸런스 확보.

### 제안 3. [V12 제안 1]과 동일 (PHASE 2 렌더링 시점에 세부 규칙 통합) ★ 추천
- 위 모든 밸런스 문제는 애초에 "도면(PHASE 2)"과 "텍스트(PHASE 3)" 내용이 다른 것에서 출발합니다. PHASE 2가 5면 시각 도면을 그릴 때 앞서 논의한 블라인드 스팟 규칙을 선제적으로 주입하면, 코너 뷰 결합 시 어떤 가중치 문제가 발생해도 무결점 형태가 도출됩니다.
- **효과:** 근본적 일관성 확보.

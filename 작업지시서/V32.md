**작성일:** 2026-04-03
**목적:** 업로드된 이미지 제어바에 '재분석' 버튼을 추가하여 분석 프로세스를 수동으로 재실행할 수 있는 기능 구현

---

## 1. 요청 사항 분석
- **위치**: 이미지 상단 부유형 제어바(Floating Control Bar)의 가장 좌측.
- **대상**: **업로드된 이미지(`item.type === 'upload'`)** 에만 표시 (생성된 이미지인 경우 미노출).
- **구성**: [재분석] | [라이브러리] | [휴지통] 순서로 배치.
- **아이콘**: `RotateCcw` (새로고침/되돌리기 아이콘).
- **기능**: 선택된 이미지에 대해 `analyzeViewpoint` 프로세스(이미지 정규화 -> 시점 분석 -> 전개도 생성)를 처음부터 다시 실행.

## 2. 변경 계획 (Technical Plan)

### A. 라이브러리 및 아이콘 추가
- `lucide-react`에서 `RotateCcw` 아이콘을 추가 임포트합니다.

### B. UI 컴포넌트 수정 (`App.tsx`)
- `canvasItems.map` 내부의 `Floating Control Bar` 섹션(line 1537 부근)을 수정합니다.
- 조건부 렌더링 적용: `item.type === 'upload'` 일 때만 버튼 노출.
- 버튼 클릭 시 `analyzeViewpoint(item.src, item.id)` 함수를 호출하여 전체 분석 파이프라인 재가동.

### C. 상태 처리
- 현재 `setIsAnalyzing(true)`를 통해 전역 로딩 상태가 관리되고 있으므로, 분석 진행 중에는 버튼을 비활성화하거나 로딩 아이콘을 표시하여 중복 실행을 방지합니다.

## 2. V9 수정이 현상을 해결하는가?

### 수정 1 (Layer C 포맷 교체) — **직접적 효과 O**
현재 Layer C가 AI에게 전달하는 것:
```
- Mass Typology: N/A
- Base Material: N/A
- Fenestration: N/A
```
수정 후 전달 내용 (5-View 구조에서 실제 값 추출):
```
- Front Geometry: A-1 Scale_X_Z, A-3 Depth_Extrusions...
- Property: B-1 Base_Color, B-2 Glass_Type...
```
→ AI가 건축 재질·구조를 실제로 이해하고 비가시권 추론 품질 향상.

### 수정 2 (Blind Spot 3단계 프로토콜, 제안 A) — **직접적 효과 O**
- **발동 조건:** 현재 각도가 09:00, 12:00, 04:30, 07:30, 10:30, 1:30 (비가시권) 일 때만
- 주입 내용: Context Typing (Type A/B) + Inside-Out Zoning + Rear/Side/Top Facade Rules
- AI가 왜 뒷면이 방화벽인지, 어디에 서비스도어가 있는지 논리적으로 추론 가능

### 결론
> **두 수정 모두 원본 이미지 재출력 현상을 직접 개선하는 효과가 있음.** 특히 Layer C → 실제 재질값이 들어가면 비가시권 생성 정밀도가 상승하고, Blind Spot 프로토콜 → AI가 뒷면/측면을 건축적으로 합리적으로 생성 가능.

## 2. 장애 요인 분석

### 위험 1 — [HIGH] contextualParamsStr JSON 내 동적 주입 시 syntax 오류

**위치**: Phase 2-3, `buildStructuredParams()` → `${contextualParamsStr}` (line 990)

```typescript
// 현재: plain text 문자열 생성
lines.push(`  Geometry: Scale=${g['A-1']?.Scale_X_Z ?? '-'}, ...`);
// → "\n  Geometry: Scale=..." 형식의 multi-line 문자열
```

**문제**: `contextualParamsStr`를 JSON code block 내부에 직접 삽입하면:
- 값 내부의 `"`, `\n`, `:` → JSON syntax 오류 발생
- AI가 생성한 AEPL 파라미터 값(예: `"Red brick, 70% opacity, rough texture"`)에 큰따옴표 포함 가능

**대응**: `buildStructuredParams()`를 `JSON.stringify()` 기반으로 재작성 → 항상 valid JSON 보장

## 1. 코드 분석 결과

### 요청 1: 아트보드 줌 독립성
- **현재 코드:** `width: ${1600 / (canvasZoom / 100)}px`, `height: ${1200 / (canvasZoom / 100)}px`
- **문제:** 줌 배율에 반비례하여 아트보드가 축소/확대됨
- **수정 방법:** 나누기 연산 제거 → `width: 1600px`, `height: 1200px` (CSS px 고정)

### 요청 2: 전개도 레이아웃 순서 변경

| | 현재 | 변경 |
|---|---|---|
| Row 0 | TOP | **REAR** |
| Row 1 | LEFT, FRONT, RIGHT | LEFT, **TOP**, RIGHT |
| Row 2 | REAR | **FRONT** |

수정 필요 항목:
- `sitePlanPrompt` 레이아웃 텍스트 지시
- `generateSitePlan` 내 Top 뷰 크롭 좌표 (현재: Row0,Col1 → 변경: Row1,Col1)
- `getElevationSlot()` 함수의 행 번호 매핑

### 요청 3: 시점-입면 매핑 수정

| 시점(Angle) | 입면 레이블 | 변경 후 그리드 좌표 (row,col) |
|---|---|---|
| 06:00 | FRONT | (2, 1) ← 기존: (1,1) |
| 12:00 | REAR | (0, 1) ← 기존: (2,1) |
| 3:00  | RIGHT | (1, 2) 유지 |
| 09:00 | LEFT | (1, 0) 유지 |

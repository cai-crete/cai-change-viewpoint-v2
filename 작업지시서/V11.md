## 3. 코드 수정 계획 (3군데)

### 수정 1 — `analysisPrompt` (bldg_ratio 필드 추가)
```json
"bldg_ratio": {
  "width": (정수, 정면 가로 비례),
  "depth": (정수, 측면 깊이 비례),
  "height": (정수, 건물 높이 비례)
}
```
→ `elevation_views` 루트 아래에 추가 요청

### 수정 2 — `generateSitePlan()` (5면 전체 크롭 저장)
현재 TOP만 크롭 → 전체 5면 크롭:
```
REAR  : row0 col1 → parameters.elevationImages.rear
TOP   : row1 col1 → parameters.elevationImages.top  (+ sitePlanImage 유지)
LEFT  : row1 col0 → parameters.elevationImages.left
RIGHT : row1 col2 → parameters.elevationImages.right
FRONT : row2 col1 → parameters.elevationImages.front
bldg_ratio → parameters.bldgRatio
```

### 수정 3 — 아트보드 JSX (CSS Grid 5패널 교체)
현재: `<img src={architecturalSheetImage}>` 1장  
변경: CSS Grid → `elevationImages.{top/front/rear/left/right}` 5개 독립 `<img>`
```
grid-template-areas: ". rear . / left top right / . front ."
grid-template-columns: depth width depth  (bldgRatio 기반)
grid-template-rows:    depth height depth
```

# System Protocol v7 및 template-layout 변경 사항 요약 (V27)

최근 업데이트된 **v7 프로토콜**과 **template-layout**의 주요 변경 사항을 기존 v6와 비교하여 정리했습니다.

## 1. 레이아웃 패러다임의 변화: "Unfolded Box"
기존의 단순 수직/수평 정렬(V26 계획)에서 물리적인 박스를 펼친 형태인 **'Unfolded Box'** 구조로 고도화되었습니다.

| 구분 | V6 (기존/V26 계획) | V7 (업데이트) | 비고 |
| :--- | :--- | :--- | :--- |
| **중앙 셀** | FRONT (정면) | **TOP (Plan/지붕)** | 기준점 변경 |
| **그리드 배치** | 상(REAR), 중(FRONT), 하(TOP) | **상(REAR), 중(TOP), 하(FRONT)** | 상하 반전 |
| **회전(Rotation)** | 0° (모든 뷰 동일) | **REAR(180°), LEFT(90°), RIGHT(-90°)** | 전개도 법칙 적용 |
| **스케일링** | `fr` (유동적 비례) | **`calc(비례 * scale)` (절대 수치)** | 결정론적 비례 고정 |
| **배경** | 불분명 (검정/흰색) | **Alpha Mandatory (100% 투명)** | 합성 호환성 강화 |
| **Overflow/Min-size** | 지정 없음 | **min-width: 0 (물리적 차단)** | 레이아웃 붕괴 방지 |

## 2. 시각적 및 기술적 요구사항 (추가 분석)
- **Alpha Channel 강제:** 모든 5면 렌더링은 배경이 없는 투명 PNG 형식이어야 합니다.
- **비율(Strict Ratio) 일치:** 생성될 이미지의 해상도 비율이 건물 절대 비례(`bldg_ratio`)와 100% 일치해야 합니다.
- **정밀 레이아웃 수치:** 
  - `gap: 60px`: 인접 뷰 간 안정적인 이격거리 확보.
  - `padding: 120px 80px`: 전체 캔버스 여백.
  - `bottom: -40px`: 라벨을 회전된 wrap 외부 '땅' 방향으로 이격 배치.
- **UI 디테일:** `border-radius: 4px`, `border: 1px solid rgba(255, 255, 255, 0.1)`, `object-fit: cover` 적용.
- **타이틀 오버레이:** `Bebas Neue` 폰트를 사용한 "IMAGE TO ELEVATION: UNFOLDED BOX" 상단 배치.

## 3. 시스템 아키텍처 및 철학 (Deep-Dive)
`protocol-image to elevation-v7`에서 새롭게 정의된 고급 건축 처리 시스템의 핵심 로직입니다.

- **DAG (Directed Acyclic Graph) 상태 기계:** 가시권(Phase 1) 데이터가 비가시권(Phase 3)을 일방적으로 지배하며 정보가 역류하지 않는 단방향 제어 게이트 구조.
- **Anti-Morphology Filter (형태 소거 필터):** Protocol B에서 텍스트 프롬프트 내에 형태/구조를 암시하는 건축 명사를 금지하고, 오직 광학/질감 수치어로만 묘사하여 미세 환각(Greebling)을 차단.
- **Reverse Flow Firewall (역류 차단 방화벽):** 내부 기능 추론 결과가 가시권의 뼈대를 오염시키는 것을 원천 차단하는 Sandbox 로직.
- **Ensemble Pair (MASTER-SLAVE):** 
  - `Master (1_Geometry)`: 픽셀의 절대 좌표를 통제 (이미지 프롬프트/ControlNet 1.0 가중치).
  - `Slave (2_Property)`: 뼈대 내부의 질감과 빛을 묘사 (텍스트 프롬프트).

## 4. V27 수행 계획 (수립 중)
- **프롬프트 고도화:** `sitePlanPrompt`에 Unfolded Box 배치 규칙 및 Alpha/Ratio 제약 조건을 주입합니다.
- **크로핑 로직 연동:** 변경된 상하 반전 구조(`top`이 중앙)에 맞춰 `cropCell` 인덱스를 재정렬합니다.
- **아트보드 UI 리팩토링:** Tailwind CSS를 사용하여 절대 수치 기반의 그리드와 `rotate-wrap` 로직을 구현합니다.

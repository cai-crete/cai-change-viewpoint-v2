## 1. 문제 A — 비율 지시 왜곡 (방안 C)

### 현황 (V37, line 966·973)

```
- Aspect Ratio: width ${panel.wUnit} : height ${panel.hUnit}
- Fill the image to the ${panel.wUnit}:${panel.hUnit} aspect ratio
```

**"Fill the image to ratio"** → AI가 1024×1024 정사각형 캔버스 전체를 지정 비율로 채우려 함
→ 건물 콘텐츠를 강제 압축/왜곡 → 원본 이미지 비율 훼손

### 변경 (방안 C) — line 966·973

```diff
- - Aspect Ratio: width ${panel.wUnit} : height ${panel.hUnit}
+ - Building proportion reference — Width : Height = ${panel.wUnit} : ${panel.hUnit}

- - Fill the image to the ${panel.wUnit}:${panel.hUnit} aspect ratio
+ - Render the building at its natural ${panel.wUnit}:${panel.hUnit} proportion, centered in the canvas
+ - Do NOT stretch or distort the building to fill the canvas; use transparent padding for the remaining area
```

**변경 후 동작:**
```
AI 인식: "건물의 Width:Height 비율 = W:H 를 참조값으로 사용"
AI 동작: 건물을 W:H 비율로 자연스럽게 렌더링, 나머지 캔버스 투명 패딩
trimAndCenter: 투명 영역 제거 → 건물 콘텐츠 중앙 정렬
```

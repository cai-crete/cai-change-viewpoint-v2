## 2. 변경 목록

### 2-A. ViewType 확장

```typescript
// 기존
type ViewType = 'eyeLevel' | 'front' | 'top';

// 변경
type ViewType = 'birdEye' | 'eyeLevel' | 'front' | 'rightSide' | 'top';
```

---

### 2-B. 신규 State 추가 (기존 states 블록에 삽입)

```typescript
const [birdEyeDirection, setBirdEyeDirection] = useState<'04:30' | '07:30'>('04:30');
const [rightSideDirection, setRightSideDirection] = useState<'03:00' | '09:00'>('03:00');
const [rightSideAltitude, setRightSideAltitude] = useState<string>('10');
```

---

### 2-C. useEffect 자동 설정 — rightSideDirection 자동 결정

기존 `selectedItemId` sync useEffect 내부, `setAnalyzedOpticalParams` 설정 직후에 추가:

```typescript
// rightSideDirection 자동 결정
const angle = item.parameters.analyzedOpticalParams?.angle;
if (angle === '04:30') setRightSideDirection('03:00');
else if (angle === '07:30') setRightSideDirection('09:00');
// 그 외 각도는 현재값 유지
```

리셋(selectedItemId 없는 경우) 블록에도 추가:
```typescript
setRightSideDirection('03:00');
```

### 2-D. SitePlanDiagram angle 매핑 업데이트

| selectedView | angle 전달값 |
|---|---|
| `'birdEye'` | `birdEyeDirection` |
| `'eyeLevel'` | `eyeLevelDirection` |
| `'front'` | `'06:00'` |
| `'rightSide'` | `rightSideDirection` |
| `'top'` | `'06:00'` |

```typescript
// 기존
angle={selectedView === 'eyeLevel' ? eyeLevelDirection : '06:00'}

// 변경
angle={
  selectedView === 'birdEye' ? birdEyeDirection :
  selectedView === 'eyeLevel' ? eyeLevelDirection :
  selectedView === 'rightSide' ? rightSideDirection :
  '06:00'
}
```

### 2-F. UI 사이드바 — View 버튼 블록 업데이트

#### 버튼 배열 (순서)
```typescript
['birdEye', 'eyeLevel', 'front', 'rightSide', 'top'] as ViewType[]
```

#### 버튼 라벨 매핑
| ViewType | 라벨 |
|---|---|
| `'birdEye'` | `조감투시뷰` |
| `'eyeLevel'` | `아이레벨투시뷰` |
| `'front'` | `정면뷰` |
| `'rightSide'` | `우측면뷰 / 좌측면뷰` |
| `'top'` | `탑뷰` |

#### 서브옵션 추가 (기존 eyeLevel/front/top 서브옵션 블록 사이에 삽입)

**birdEye 선택 시 — 04:30 / 07:30 토글:**
```tsx
{selectedView === 'birdEye' && (
  <div className="flex gap-2 mt-1">
    {(['04:30', '07:30'] as const).map((dir) => (
      <button
        key={dir}
        onClick={() => setBirdEyeDirection(dir)}
        className={`flex-1 py-1 font-mono text-[10px] border transition-all ${
          birdEyeDirection === dir
            ? 'border-black dark:border-white bg-black text-white dark:bg-white dark:text-black'
            : 'border-black/20 dark:border-white/20 hover:border-black dark:hover:border-white'
        }`}
      >
        {dir}
      </button>
    ))}
  </div>
)}
```

**rightSide 선택 시 — 03:00 / 09:00 토글 + Altitude 입력:**
```tsx
{selectedView === 'rightSide' && (
  <>
    <div className="flex gap-2 mt-1">
      {(['03:00', '09:00'] as const).map((dir) => (
        <button
          key={dir}
          onClick={() => setRightSideDirection(dir)}
          className={`flex-1 py-1 font-mono text-[10px] border transition-all ${
            rightSideDirection === dir
              ? 'border-black dark:border-white bg-black text-white dark:bg-white dark:text-black'
              : 'border-black/20 dark:border-white/20 hover:border-black dark:hover:border-white'
          }`}
        >
          {dir}
        </button>
      ))}
    </div>
    <div className="flex items-center gap-2 mt-1">
      <span className="font-mono text-[10px] opacity-60 uppercase tracking-widest whitespace-nowrap">Altitude (m)</span>
      <input
        type="number"
        value={rightSideAltitude}
        onChange={(e) => setRightSideAltitude(e.target.value)}
        className="flex-1 border border-black/30 dark:border-white/30 bg-transparent font-mono text-xs py-1 px-2 text-right focus:outline-none focus:border-black dark:focus:border-white"
        min="0"
      />
    </div>
  </>
)}
```
